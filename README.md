# Advent of Code 2024

## Principle

- I do it using Rust.
- No dependencies except the standard library.
- Good performance if I can.
- Do not promise I will finish all problems.
- Just for fun.

## Run

```bash
cargo test --test day1 --release 
```

Remove `--test day1` if you want run all of them.
